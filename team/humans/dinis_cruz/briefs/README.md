# Briefs — HUMAN-ONLY

**Agents: do NOT create files in this folder under any circumstances.**

This is the human's raw inbox — memos, voice-memo transcripts, half-formed ideas, raw brain-dumps. Nothing here is yet a triaged task.

## Agent Workflow

1. Read briefs here to understand intent.
2. If a brief describes work the team should do, produce a **triaged brief** under `team/comms/briefs/v{version}__{name}.md` that cites the source memo by path.
3. All subsequent role work references the triaged brief, not this raw one.

## Why Read-Only?

Anything you write here would be confused with the human's own voice. That breaks the provenance chain. The triaged brief in `team/comms/briefs/` is where agent-authored content belongs.
