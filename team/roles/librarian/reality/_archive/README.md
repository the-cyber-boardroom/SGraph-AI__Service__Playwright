## How to use

- **Reading the truth:** start at [`../index.md`](../index.md). Do NOT cite anything in this folder.
- **Tracing history:** use these files to see what was known at a given version line.
- **Migrating remaining domains:** the per-file sources for the 9 unmigrated domains live in [`v0.1.31/`](v0.1.31/). When M-003 lifts the last domain out, this folder stops being a "shim" and becomes pure history.
