# Cross-Role Briefs

Human briefs that have been triaged and turned into cross-role work live here. Direct-from-human briefs (before triage) live in `team/humans/dinis_cruz/briefs/` and are HUMAN-ONLY — agents never write there.

## Naming

`v{version}__{description}.md`

## Lifecycle

1. Human writes a memo in their own folder.
2. Librarian or Architect reads it, classifies scope, and produces a triaged brief here referencing the source.
3. Roles execute; reviews and debriefs cross-reference this brief.
4. When delivered, move to `archive/` with the closing-commit hash appended to the filename.
