## Scope

Closes the second half of the `records check` UX surface. The default mode (`--authoritative`) shipped in P1 is cache-pollution-safe; the opt-in modes (`--public-resolvers`, `--local`, `--all`) ship here with explicit warnings and `[y/N]` confirms.

| Flag | What it does | Cache impact |
|------|--------------|--------------|
| (none) | Authoritative NS direct via `dig @ns +norecurse`. | Zero pollution. |
| `--public-resolvers` | + fan-out across 8 curated public resolvers. | Each resolver caches the answer for the record's TTL. **Locks in staleness** for future updates within the TTL window. |
| `--local` | + `dig` against the host's default resolver. | Local resolver (often a corporate proxy / VPN) caches; no portable flush. |
| `--all` | All three. | Both pollutants. |

`--yes` skips the prompt; `--min-resolvers N` controls the public-resolver quorum (default 5/8); `--json` emits a combined result payload. Exit codes: `0` all pass, `1` auth fail (highest priority), `2` public-resolver quorum fail, `3` local mismatch.
