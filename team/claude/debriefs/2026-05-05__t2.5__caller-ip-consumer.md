# Debrief — T2.5 caller-IP consumer

**Date:** 2026-05-05
**Status:** PARTIAL — backend endpoint not yet shipped; Option B heuristic in place

---

## What shipped (Option B)

`sg-compute-launch-form._seedCallerIp()` implements a local heuristic:

- `localhost` / `127.0.0.1` / `0.0.0.0` → sets `127.0.0.1`
- Remote host → leaves field empty; helper text + "Find my public IP ↗" link to Google

The form includes a "Caller IP" input in the Advanced section (collapsed by default) with a hint line explaining why it is needed.

ipify.org was removed in FV2.11 (no third-party dashboard calls policy). Option B preserves privacy while keeping the form functional for local development — the most common use case.

---
