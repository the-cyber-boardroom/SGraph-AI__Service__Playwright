# Debrief — T2.3 a11y keyboard nav + contrast fixes

**Date:** 2026-05-05
**Status:** COMPLETE
**Commit:** ae5fca2

---

## What shipped

1. **Roving tabindex keyboard nav on `sg-compute-nodes-view`.**
   Added `_initTabKeyNav()` called in `onReady()`. The method sets `tabindex="0"` on the active tab and `tabindex="-1"` on all others. A `keydown` listener on `[role="tablist"]` handles ArrowLeft, ArrowRight, Home, End — cycling focus and activating tabs. Uses `this.shadowRoot?.activeElement` (required for Shadow DOM; plain `document.activeElement` would always be the host element).

2. **WCAG AA contrast fix on `--text-3` token.**
   Old value `#5a6478` failed at 2.8:1 on `--bg-elev: #162040`. New value `#96a6bc` achieves:
   - 4.65:1 on `--bg-elev` (lightest dark surface)
   - 5.22:1 on `--bg-panel`
   - 6.04:1 on `--bg-app`
   All pass AA (≥4.5:1). Single token change in `shared/ec2-tokens.css` — no splitting needed.

3. **`:focus-visible` ring on `.sgl-tab`** added to `ec2-tokens.css` for keyboard discoverability.
