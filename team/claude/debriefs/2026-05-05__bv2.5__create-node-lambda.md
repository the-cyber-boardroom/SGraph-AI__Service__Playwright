## Key decisions

- **`spec_id` dispatch in EC2__Platform**: `_create_docker_node` is the only live spec (docker). Other spec IDs raise `NotImplementedError`. Adding podman/vnc is a future phase once those services have a `create_stack` equivalent.
- **`Platform.create_node` default `spec=None`**: The `spec: Schema__Spec__Manifest__Entry` parameter is kept for forward compatibility but defaulted to `None` — the route doesn't need to pass a manifest since `request.spec_id` carries that information.
- **Lambda handler is copy of host-plane pattern**: Same try/except ImportError guard for Mangum. `handler = None` when Mangum not installed — uvicorn uses `_app` directly.
- **Route test via Fake__Platform**: EC2 integration path (real Docker__Service.create_stack → real boto3) is tested only with live AWS. All unit tests use Fake__Platform, consistent with the zero-mock pattern.
