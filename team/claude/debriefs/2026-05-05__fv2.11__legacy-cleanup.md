## Reasoning

**stacks-pane removal:** `sp-cli-stacks-pane` was superseded by `sp-cli-nodes-view` in FV2.2. No `<sp-cli-stacks-pane>` element was inserted into the DOM in admin.js or any HTML file — the `querySelector` always returned null (the optional chain made it a safe no-op). The component was being loaded and parsed needlessly.

**ipify.org removal:** The `api.ipify.org` call leaked the operator's IP address to a third-party service on every launch-form open, without any disclosure. The displayed IP was informational only (the backend determines the actual security group source IP from the HTTP request). The explanatory note in the Advanced section is retained; the IP value row is removed.
