## How

Pure decorator move. Function bodies and signatures unchanged.

```python
# scripts/provision_ec2.py — right after `app = typer.Typer(...)`
pw_app = typer.Typer(no_args_is_help=True,
                       help='Playwright + agent-mitmproxy stack (the headless API).')
app.add_typer(pw_app, name='playwright')
app.add_typer(pw_app, name='pw',  hidden=True)

vault_app = typer.Typer(...)
pw_app.add_typer(vault_app, name='vault')                                         # was app.add_typer
pw_app.add_typer(vault_app, name='v',     hidden=True)

ami_app = typer.Typer(...)
pw_app.add_typer(ami_app, name='ami')                                             # was app.add_typer
```

Then mechanical sed: every `@app.command(...)` → `@pw_app.command(...)`. 21 decorators rewired.

## Top-level `app` help string updated

Was: `'Manage the Playwright + agent_mitmproxy EC2 stack.'`
Now: `'SG Playwright CLI — manage ephemeral EC2 stacks across all sister sections.'`

The CLI is no longer "the Playwright provisioner" — it's a multi-section stack orchestrator.
