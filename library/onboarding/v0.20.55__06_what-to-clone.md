# What to Clone

**Version:** v0.20.55

Clone these repos as **read-only references**. Do not modify them.

---

## 1. SGraph-AI__App__Send (main repo)

```bash
git clone --depth 1 -b dev https://github.com/the-cyber-boardroom/SGraph-AI__App__Send /tmp/sgraph-send-ref
```

**Read these files:**

| File | Why |
|---|---|
| `.github/workflows/ci-pipeline.yml` | CI pipeline pattern (layered workflows) |
| `.github/actions/aws__deploy__lambda/action.yml` | Deploy-via-pytest action |
| `tests/deploy/Admin__Lambda/deploy_aws/test_Deploy__Admin__Service__base.py` | Deploy test base class pattern |
| `sgraph_ai_app_send/lambda__admin/fast_api/Fast_API__SGraph__App__Send__Admin.py` | Serverless__Fast_API usage |
| `sgraph_ai_app_send/lambda__admin/lambda_function/deploy/Deploy__Service.py` | Deploy__Serverless__Fast_API |
