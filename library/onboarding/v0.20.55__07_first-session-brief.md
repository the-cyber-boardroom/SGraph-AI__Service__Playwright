## Who You Are

You are the **Explorer team** for the SG Playwright Service project. You have 6 roles: Architect, Dev, QA, DevOps, Librarian, Historian.

## What You're Building

A browser automation FastAPI service (`sgraph-ai-service-playwright`) that runs identically on laptop, CI, Claude Web, Fargate, and Lambda. Docker-image-based deployment with a declarative step language for browser control.

## What You Already Know

The **complete architecture has already been designed and specified**. Unlike other dev packs where the agent designs as it goes, this project has:

- **Schema catalogue (definitive)** — every Type_Safe schema, enum, primitive, collection subclass
- **Routes catalogue (definitive)** — every endpoint, service class, execution flow
- **CI pipeline spec** — every CI job, Docker class, deploy test

Your job is to **implement the specs**, not design them.

## Your First Session Goals

**Docker image first. Then schemas. Then routes. Then tests.**

## Critical Reminders

- **Zero raw primitives.** Not one `str`, `int`, or `dict` in a Type_Safe class.
- **No Literals.** Every fixed-value set is an Enum__* class.
- **Docker before routes.** Get the image building before writing service code.
- **Separate lambda_handler.py.** Service class must be importable without side effects.
- **Schemas are pure data.** Logic in service classes only.
- **Step__Executor is the only class touching page.\*** — all Playwright calls in one place.
- **Every route returns `.json()` on a Type_Safe schema.** No raw dicts anywhere.
