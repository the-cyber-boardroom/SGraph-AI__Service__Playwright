## Mission

Build `sgraph-ai-service-playwright` — a browser automation FastAPI service that runs identically on laptop, CI, Claude Web, Fargate, and Lambda. Docker-image-based. Declarative step language. Vault-integrated. Zero raw primitives.

## Non-Negotiable Rules

1. **Zero raw primitives** — no `str`, `int`, `float`, `list`, `dict` in Type_Safe classes
2. **All classes extend Type_Safe** — no plain Python classes
3. **No Pydantic** — ever
4. **No boto3** — ever (use osbot-aws)
5. **No Literals** — all fixed-value sets use proper Enum__* classes
6. **Schemas are pure data** — no methods on schema classes
7. **Docker image before service code** — Dockerfile + CI pipeline must work before routes
8. **Separate lambda_handler.py** — service class importable without side effects
9. **Every route returns `.json()` on a Type_Safe schema** — no raw dicts
10. **Deploy-via-pytest** — deployment steps are pytest tests, not shell scripts
11. **Step__Executor is the ONLY class calling page.\*** — all Playwright API calls in one place
12. **Per-action capture validation** — not at session create
