# _to_process — Inbox

**Purpose:** Staging area for raw material that has not yet been classified into `docs/specs/`, `docs/research/`, `guides/`, or elsewhere.

## How it works

1. A contributor (human or agent) drops a document here when they don't yet know where it belongs.
2. The Librarian reads each new document, classifies it, and moves it to the correct subdirectory with the appropriate version prefix.
3. Cross-references are added from related documents.
4. The document is removed from this folder once processed.

**Keep this folder small.** If material is piling up, the Librarian flags it in the next master index.
